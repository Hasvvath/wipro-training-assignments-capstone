# EduLearn Course Management App

## How to Run
1. npm install
2. ng serve
3. Open http://localhost:4200

## Features
- Course list display
- View course details
- Edit course title dynamically

## Concepts Used

Property Binding:
Used to pass selected course from CourseList to CourseDetail

Event Binding:
Used to handle button click

Two-Way Binding:
Used with ngModel to update title in real-time
